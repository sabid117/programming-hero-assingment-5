Question-1: What are the different ways to select an element in the DOM?

  1. getElementById
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

1. innerText changes the text content of an element
2. innerHTML changes the html contents of an element
3. textContent is the text inside an element, excluding any HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a way of adding event listeners to their parents instead of multiple child elements. It uses event bubbling.

Question-4: What is event bubbling in the DOM?

When a event is triggered Event bubbling propagates it up through it's parent elements until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

  1. Elements can be created using document.createElement()
  2. Elements can be added using append() or appendChild().
  3. Elements can be removed using removeChild().
